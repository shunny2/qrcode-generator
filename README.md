## How to Run

  First, start by cloning the repository:
  
  ```shell
  git clone https://github.com/shunny2/qrcode-generator
  ```
  
  Then, enter the project folder and install the dependencies:
  ```bash
  npm install
  ```
  
  Finally, run the project:
  ```bash
  npm run dev
  ```
